# KPIs — cymobility

## Métriques cœur
- **DAU/MAU** ratio (cible: >0.3)
- **Projets complétés / utilisateur / semaine**
- **Temps moyen objectif → livrable**
- **NPS** post-livrable

## Métriques business
- **MRR** mensuel
- **Churn** mensuel (cible: <5%)
- **CAC** sur Product Hunt vs ads
- **LTV/CAC** > 3 pour scaler

## Dashboard recommandé
Metabase ou Posthog. Tracker chaque tool_call agent pour mesurer le coût LLM par projet.
